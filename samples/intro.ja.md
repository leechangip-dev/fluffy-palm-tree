# 翻訳自動化ツールのご紹介

このツールはClaude APIを活用して、さまざまなドキュメントを複数の言語に自動翻訳します。

## 主な機能

- **多言語対応**: 韓国語、英語、日本語、中国語、スペイン語など10言語に対応
- **ファイル形式**: Markdown、テキスト、JSONファイルの翻訳
- **一括翻訳**: 1つのファイルを複数の言語に同時翻訳
- **コード保持**: コードブロック内の内容は翻訳せずそのまま維持

## 使用例

```bash
# 単一言語翻訳
python src/cli.py file samples/intro.md -t en

# 一括翻訳
python src/cli.py file samples/intro.md -t en,ja,zh

# テキスト直接翻訳
python src/cli.py text "안녕하세요" -t en
```

## 参考事項

翻訳品質を向上させるには、`-c` オプションでドキュメントのコンテキスト情報を追加で提供することができます。

## GitHub Actions 自動翻訳

このファイルを修正してpushすると、GitHub Actionsが自動的に英語、日本語、中国語の翻訳ファイルを生成します。

## 対応ファイル形式

Markdown、テキスト、JSON、YAML形式をすべてサポートしています。

## 自動化の効果

繰り返しの翻訳作業を自動化することで、時間とコストを削減できます。GitHub Actionsと連携することで、ファイルの修正直後に翻訳が実行されます。

## 注意事項

- APIキーが外部に漏洩しないよう、GitHub Secretで管理してください。
- 翻訳結果ファイル（`.en.md`、`.ja.md` など）は直接修正しないでください。ソースファイルの修正時に自動的に上書きされます。
- 大容量ファイルは翻訳に時間がかかる場合があります。
