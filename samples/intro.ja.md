# 翻訳自動化ツールのご紹介

このツールはClaude APIを活用して、さまざまな文書を複数の言語に自動翻訳します。

## 主な機能

- **多言語サポート**: 韓国語、英語、日本語、中国語、スペイン語など10言語に対応
- **ファイル形式**: Markdown、テキスト、JSONファイルの翻訳
- **一括翻訳**: 1つのファイルを複数の言語に同時翻訳
- **コードの保持**: コードブロック内の内容は翻訳せずそのまま維持

## 使用例

```bash
# 単一言語への翻訳
python src/cli.py file samples/intro.md -t en

# 一括翻訳
python src/cli.py file samples/intro.md -t en,ja,zh

# テキストの直接翻訳
python src/cli.py text "안녕하세요" -t en
```

## 参考事項

翻訳品質を向上させるには、`-c` オプションで文書のコンテキスト情報を追加で提供することができます。
